# PoudlardPlugin
Plugin test pour PoudlardRP

Le plugin est en 1.15.2 Spigot, lors de la connection d'un joueur, une baguette de sorcier lui est donnée (stick).
Il a alors la possiblilté d'utiliser la commande : /spell avada ou expel pour choisir quel sort lancer avec cette baguette. 
Le sort avada kedavra lance une flèche verte qui lorsqu'elle touche une entité la tue directement. 
Le sort expelliarmus lance aussi flèche cette fois rouge mais ne tue et pousse l'entité plus loin. 

Pour les deux sorts, j'ai utilisé une entité flèche, des particules (de couleurs vertes ou rouges) et un son apparaissent lors du lancé et du hit.
